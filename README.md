# 🌸 Mood Tracker System

A modern mental health & productivity tracker system built using CakePHP, Bootstrap, Chart.js and MySQL.

---

## ✨ Features
- Mood Tracking
- Pomodoro Timer
- Task Management
- Dashboard Analytics
- Responsive Design
- Animated UI

---

## 🖥️ Technologies Used
- CakePHP
- PHP
- Bootstrap 5
- MySQL
- Chart.js
- JavaScript
